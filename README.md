# Lucky Orange template for Google Tag Manager

This template allows website owners to easily place the Lucky Orange tracking code on all pages of their website without the need for a developer.

:warning: **Heads up!** :warning:

This template only works with the new version of Lucky Orange. Users on [Lucky Orange Classic](https://classic.luckyorange.com/) should use the [unoffical template](https://tagmanager.google.com/gallery/#/owners/luratic/templates/LuckyOrange) for GTM.
